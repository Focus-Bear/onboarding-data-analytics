# VS Code Productivity Shortcuts

## Navigation
- Ctrl + P: Quick open files
- Ctrl + Shift + O: Go to symbol
- Ctrl + Tab: Switch editor tabs

## Editing
- Alt + Click: Multiple cursors
- Ctrl + D: Select next occurrence
- Ctrl + Shift + K: Delete line

## Debugging
- F5: Start debugging
- F10: Step over

## Terminal
- Ctrl + `: Toggle terminal
- Ctrl + Shift + C: Copy from terminal

## Reflection
Most useful shortcuts for me: multiple cursors and quick file nav (Ctrl + P, Alt + Click).  
They save time switching contexts and editing in bulk.  
Keyboard shortcuts improve productivity by reducing friction between thinking and typing.
