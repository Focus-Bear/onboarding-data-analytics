# Git Knowledge

## Experience
I’ve used Git previously for university group projects and personal repositories. However, this internship is my first time applying Git workflows in a more professional, collaborative setting—especially through GitHub pull requests.

## Git Client Chosen
I chose to stick with the **command line interface (CLI)** via macOS Terminal, as it offers the most control and is ideal for learning. I’ve also explored **Git integration in VS Code**, which is helpful for visualising changes and staging commits quickly.

## Most Interesting Learning
The most interesting thing I learned today was how Git tracks and stores changes using snapshots (rather than deltas) and how branches are lightweight references. Also, understanding how to use `git checkout <branch> -- <file>` to selectively bring files from another branch was a useful real-world tip during onboarding.
