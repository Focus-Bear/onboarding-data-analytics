# Chrome Extensions for Developer Productivity

## Extensions Installed

1. **React Developer Tools**
   - Helps inspect React component hierarchies and props/state.
   - Useful for understanding how components are nested and how state flows.

2. **Redux DevTools**
   - Allows inspection of Redux state changes and time-travel debugging.
   - Critical when working with apps using Redux for state management.

3. **JSON Viewer**
   - Pretty-prints raw JSON responses from APIs in the browser.
   - Makes API responses readable and easier to debug.

4. **Lighthouse**
   - Used for auditing web app performance, accessibility, and SEO.
   - Helped me understand key metrics that affect real-world performance.

## Key Learning
The most useful insight was seeing how **Redux DevTools** allows time-travel debugging — being able to replay past states helped me understand how user interactions affect app logic step-by-step.
