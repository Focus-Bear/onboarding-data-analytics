# Terminal Client Setup & Reflections

## Terminal Client Chosen
I chose to use the built-in **macOS Terminal** paired with **zsh**, as it integrates well with the Mac environment and has rich customization support. I considered iTerm2 but decided to start simple.

## Customizations
- Switched shell to `zsh` for better autocomplete and plugin support.
- Added the `agnoster` theme via Oh My Zsh for a cleaner prompt.
- Added basic aliases like:
  - `gs` for `git status`
  - `ga` for `git add`
  - `gcmsg` for `git commit -m`

## Most Useful Command Learned
```bash
cd -
```
This command toggles between the last two directories I visited. Super helpful for jumping back and forth between project folders.
