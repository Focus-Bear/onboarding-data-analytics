# AI Tools for Development

## Tools Tried
- **GitHub Copilot:** Integrated in VS Code for autocomplete-style code suggestions.
- **ChatGPT:** Used for asking coding questions and debugging logic issues.
- **Claude AI:** Used as an alternative perspective to compare suggestions with ChatGPT.

## What Worked Well
- GitHub Copilot was extremely helpful for boilerplate code and repetitive patterns (like loops, conditionals).
- ChatGPT helped debug a tricky Python error by explaining the issue step-by-step.
- Claude AI was useful for summarising concepts more concisely.

## What Didn't Work Well
- Copilot sometimes suggested incorrect logic for business-specific functions.
- AI tools can hallucinate – it's important to verify their output carefully.

## When AI is Most Useful
- Writing repetitive code blocks or boilerplate.
- Refactoring or explaining unknown code.
- Debugging syntax errors and simple logic problems.
